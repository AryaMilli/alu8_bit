# alu8_bit
8 bit ALU in Verilog with a testbench to simulate and to display the values on console
The select function varies as follows: 

| sel (3-bit) | Operation       |
|-------------|-----------------|
| 000         | Addition        |
| 001         | Subtraction     |
| 010         | Rotate Left     |
| 011         | Rotate Right    |
| 100         | AND             |
| 101         | NAND            |
| 110         | NOR             |
| 111         | OR              |
---------------------------------
